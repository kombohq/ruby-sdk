# PostAtsApplicationsApplicationIdAttachmentsRequestBodyOverrideDocumentCategory

Allows you to override the document category for the attachment.

## Example Usage

```ruby
require "kombo"

value = PostAtsApplicationsApplicationIdAttachmentsRequestBodyOverrideDocumentCategory::IRC_CANDIDATE_RESUME
```


## Values

| Name                        | Value                       |
| --------------------------- | --------------------------- |
| `IRC_CANDIDATE_RESUME`      | IRC_CANDIDATE_RESUME        |
| `IRC_CANDIDATE_COVERLETTER` | IRC_CANDIDATE_COVERLETTER   |
| `MISC`                      | MISC                        |
| `IRC_INTERNAL`              | IRC_INTERNAL                |
