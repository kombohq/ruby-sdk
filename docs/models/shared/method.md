# Method

The HTTP method (e.g., `GET`) of the request.

## Example Usage

```ruby
require "kombo"

value = Method::GET
```


## Values

| Name     | Value    |
| -------- | -------- |
| `GET`    | GET      |
| `POST`   | POST     |
| `DELETE` | DELETE   |
| `PUT`    | PUT      |
| `PATCH`  | PATCH    |
