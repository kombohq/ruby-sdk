# PostAtsCandidatesRequestBodyPeriod

The period of the salary expectations. Must be one of `MONTH` or `YEAR`.

## Example Usage

```ruby
require "kombo"

value = PostAtsCandidatesRequestBodyPeriod::MONTH
```


## Values

| Name    | Value   |
| ------- | ------- |
| `MONTH` | MONTH   |
| `YEAR`  | YEAR    |
