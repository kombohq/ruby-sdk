# SyncFinishedWebhookPayloadIntegrationCategory

The category of the integration (HRIS, ATS, ASSESSMENT, or LMS).

```ruby
require "kombo"

value = SyncFinishedWebhookPayloadIntegrationCategory::HRIS
```
