# FieldScopeConfigSetting

The setting of the datapoint in the scope config that you configured in the Kombo dashboard.

## Example Usage

```ruby
require "kombo"

value = FieldScopeConfigSetting::ENABLED
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ENABLED`  | ENABLED    |
| `DISABLED` | DISABLED   |
| `OPTIONAL` | OPTIONAL   |
