# PostHrisAbsencesPositiveResponseUnit

The unit of time for this absence. Can be `HOURS` or `DAYS`.

## Example Usage

```ruby
require "kombo"

value = PostHrisAbsencesPositiveResponseUnit::HOURS
```


## Values

| Name    | Value   |
| ------- | ------- |
| `HOURS` | HOURS   |
| `DAYS`  | DAYS    |
